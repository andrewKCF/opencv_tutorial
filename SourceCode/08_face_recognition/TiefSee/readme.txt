
名稱
    TiefSee (舊名AeroPic)

版本
    3.0.2

說明
    免安裝的圖片檢視器，真正專注於看圖的軟體
    簡約高雅的界面，不因功能強大就在界面上攤一堆使用者看不懂的按鈕
    支援多種特殊格式：apng、svg、psd、raw、web、pixiv動圖、exe 等...
    使用大量瀏覽模式，讓看漫畫輕鬆無比 
    將視窗固定在最上層，協助完成複雜工作
    在資料夾或桌面長按空白鍵，快速預覽圖片

運行需求
    Microsoft .NET Framework 4.6.2 以上(含)
    https://www.microsoft.com/zh-TW/download/details.aspx?id=56116
